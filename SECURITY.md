# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue
2. Use [private vulnerability reporting](https://github.com/KyleNesium/AIBattery/security/advisories/new) on GitHub
3. Include steps to reproduce, if possible
4. Allow reasonable time for a fix before public disclosure

## Scope

AI Battery stores OAuth tokens in the macOS Keychain and reads local Claude Code session logs. Security concerns related to token handling, credential storage, or unintended data exposure are in scope.

## Response

Critical issues will be patched and released as soon as possible.
